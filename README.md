# 💰 Smart Finance Assistant

AI-powered personal finance manager using 
Google Gemini AI and Google Services.

## Chosen Vertical
**Finance** - Personal Finance Management Assistant

## Features
- 🤖 AI Financial Advisor (Gemini AI)
- 💳 Smart Expense Tracker
- 📊 Budget Analysis
- 🌍 Real-time Currency Converter
- 📈 Investment Recommendations
- 💡 Financial Health Score

## Google Services Used
- ✅ Google Gemini AI (Financial advice)
- ✅ Google Exchange Rate API (Currency)
- ✅ Google Generative AI (Analysis)

## Tech Stack
- Python Flask (Backend)
- Google Gemini Pro (AI)
- HTML/CSS/JavaScript (Frontend)
- Exchange Rate API (Currency)

## Setup Instructions

### 1. Clone Repository
git clone https://github.com/USERNAME/smart-finance-assistant.git
cd smart-finance-assistant

### 2. Install Requirements
pip install -r requirements.txt

### 3. Setup Environment Variables
Create .env file:
GEMINI_API_KEY=your_key_here
EXCHANGE_API_KEY=your_key_here

### 4. Run Application
python app.py

### 5. Open Browser
Go to: http://localhost:5000

## How It Works
1. User asks financial questions
2. Gemini AI provides expert advice
3. User tracks income and expenses
4. AI analyzes spending patterns
5. System provides budget recommendations
6. Currency converter for global finance

## Approach and Logic
- Context-aware AI responses
- Real-time expense tracking
- Financial health scoring
- 50/30/20 budget rule implementation

## Assumptions
- Internet connection required
- Valid API keys needed
- Python 3.8+ required
- Modern browser required
